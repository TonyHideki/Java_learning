\# Java Learning Repository



\## 📘 概要

Java基礎文法の習得と、オブジェクト指向プログラミング理解のための練習リポジトリです。



\## 🧩 開発内容

\- 計算機プログラム（ループなし／ループあり／前回結果(ans)対応）

\- 現在は `CalculatorWithAns` クラスでリファクタリングを進行中



\## 🧠 学んだこと

\- Scannerを用いたコンソール入力の扱い

\- 制御構文（if文・while文・switch文）

\- メソッド分割と例外処理の設計

\- 変数スコープと状態保持(ans機能)
